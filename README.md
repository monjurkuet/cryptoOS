# CryptoData APIs

Documentation and scripts for various cryptocurrency data APIs.

## Projects

### [CBBI](cbbi/)
ColinTalksCrypto Bitcoin Bull Run Index - Bitcoin on-chain indicators

- MVRV Z-Score
- Reserve Risk
- RHODL Ratio
- Coin Metrics data

### [Hyperliquid](hyperliquid/)
Hyperliquid DEX API documentation and data collection scripts

- Trading data (tickers, orderbooks, candles)
- Vaults information
- User activity metrics
- Funding rates, open interest, liquidations

### [CryptoQuant](cryptoquant/)
CryptoQuant on-chain analytics documentation

- Market indicators
- Exchange flows
- Derivatives data
- Fund data (premiums, market cap)

### [Bitcoin Magazine Pro](bitcoinmagazinepro/)
Bitcoin Magazine Pro market indicators

- Fear & Greed Index
- MVRV Z-Score
- Puell Multiple
- Market metrics

## Usage

Each folder contains:
- `README.md` - API documentation
- `docs/` - Detailed endpoint documentation
- `scripts/` - Download and data collection scripts
- `data/` - Downloaded data (gitignored)
